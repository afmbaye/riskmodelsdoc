Modèle de données

L'organisation des données repose sur une modélisation relationnelle structurée. Il permet de relier différentes sources d’information dans une structure cohérente. Chaque table joue un rôle bien défini, et les relations entre elles permettent de croiser efficacement les informations nécessaires aux visualisations et aux KPIs.

Sources de données

Le modèle est alimenté par 3 principales sources de données, mises à jour quotidiennement pour garantir une analyse en temps réel.

14 reportings [Suivi des Échéances]

2 fichiers pour chaque filiale (7 filiales) soit 14 fichiers au total : Celui de la fin du mois précédent (30, 31 ou 28 si février) + celui à la date du jour

mapping.xlsx

Fichier de correspondance pour le mapping des données agences, filiales, segments,...

Photo pays

Dossier contenant les pictos de drapeaux des 7 différentes filiales pour les visualisations

Schéma relationnel

Vue d'ensemble des relations entre les différentes tables du modèle.

Description des tables

Tables de faits

Elles regroupent les événements clés à analyser. Ici, chaque ligne représente une échéance.

MAIN VINTAGE ECHEANCE

Table de faits principale contenant l'historique détaillé de toutes les échéances de remboursement. Chaque ligne représente une échéance avec son statut, son montant et ses dates associées.

MAIN CONTRACT EOM

Table de faits contenant les données agrégées des contrats en fin de mois (End of Month). Elle permet l'analyse des positions mensuelles et l'évolution des encours.

Tables de dimension

Elles jouent le rôle de référentiel et enrichissent les tables de faits.

New Segment

Référentiel de la segmentation: Nano, SE,...

Dateday

Table de dates utilisée pour l'analyse temporelle.

Mapping number month

Mapping du mois sous format numérique.

FX_rates

Référentiel des taux de change pour la conversion des devises.

Main DAO name

Dictionnaire des gestionnaires de portefeuille (DAO).

Photos pays

Banque d'images pour les pictos visuels par pays.

Code_agence

Liste des agences avec leurs codes.

Countries

Dictionnaire pays avec région, code et devise.

Main products

Dictionnaire des produits.
