Metrics

DESCRIPTION

Cette section fournit un guide de référence pour tous les KPIs utilisés dans le modèle. Chaque KPI est décrit afin de faciliter la compréhension de ce qu'il mesure, comment il est calculé et où le retrouver dans le modèle.

Note: Il y'a une version EUR et Local Currency (Monnaie locale) pour tous les KPIs montants.
